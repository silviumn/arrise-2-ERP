# Conversie comenzi lentile · Arrise → Neomanager

Aplicație web pentru convertirea automată a tabelului de comenzi exportat din Arrise în formatul cerut de Neomanager.

## Lentile suportate

**Sferice**: Biofinity · Avaira Vitality · Clariti 1 day · Energys
**Torice**: Biofinity Toric · Avaira Vitality Toric · Clariti 1 day Toric · Proclear Toric

## Stack

- Python 3.10+
- Streamlit
- openpyxl
- pandas

## Local

```bash
pip install -r requirements.txt
streamlit run app.py
```
